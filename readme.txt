Instrucciones para ejecutar:

1.- Compilar con 'make'

2.- Ejecutar el binario con './setcover n m threads'.
    n: número de conjuntos
    m: número de elementos
    threads: número de hilos para la paralelización

3.- Para imprimir los arreglos de bits cambiar el parámetro 'PRINT'.


make

./setcover 1000000 128 4
